PROCEDURE

Task Augmentation

  • The inverse kinematics (IK) solver is used to compute an initial joint configuration that achieves the desired end-effector pose specified by the input parameters.
  • The function augments the initial joint configuration by adjusting the angle of the first joint.
  • Later the function constructs a new homogeneous transformation matrix that retains the original end-effector pose. This ensures that the desired pose remains the same despite the adjustment in the joint angle.
  • The function re-solves the IK problem to find a new joint configuration that achieves the same end-effector pose while considering the adjusted first joint angle.

Pick and Place

  • The robot model is loaded from a URDF file and its properties are configured.Pick and place positions are defined as homogeneous transformation matrices.
  • The IK solver is used to compute joint configurations for both the pick and place positions.
  • Linear interpolation is performed between the joint configurations of the pick and place positions to generate intermediate joint configurations.
  • These interpolated joint configurations form a trajectory for the robot to move smoothly from the pick to the place position.
  • Task augmentation is applied to adjust the first joint angle of the robot's configuration for both the pick and place positions.
